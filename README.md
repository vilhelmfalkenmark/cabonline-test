# Vilhelm Falkenmark Cabonline Test

## Installation

1. Have node version 10.13 and yarn version 1.13. installed
2. Run

```sh
yarn
```

to install node_modules.

3. Run

```sh
yarn dev
```

to start dev mode on http://localhost:1337

4. To run linting run

```sh
   yarn lint
```

5. For typechecking run

```sh
   yarn type-check
```
